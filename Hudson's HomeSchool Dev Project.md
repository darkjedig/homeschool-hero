# **Project Brief: Gamified Homeschool Learning Platform**

I want to build a permanently live, cloud-backed homeschool web application for my son, who is around UK Year 6 / US Grade 6 level, suitable for an 11 to 12-year-old learner.

The app should combine academic learning, technical skills, practical life skills, quizzes, gamification, progress tracking, rewards, and parent-controlled lesson management into one polished platform.

The goal is to create a homeschool system that feels more like a premium interactive learning game than a basic education dashboard. It should be clean, fast, secure, fun, highly visual, and easy for both parent and student to use.

I want you to start by creating a full project plan for this app, including the suggested tech stack, database structure, user flows, page structure, UI/UX design system, core features, build phases, and any additional concepts or subjects that should be included for an 11 to 12-year-old based on the class types listed below.

---

## **Main Goal**

Build a cloud-backed homeschool web app that stays online 24/7 and does not go to sleep.

The platform should allow the student to:

* Watch video lessons  
* Read short lesson notes  
* Complete quizzes  
* Play learning games  
* Earn points  
* Unlock badges  
* Complete a mandatory review quiz every Friday  
* Spend points in a parent-managed reward shop

The platform should allow the parent to:

* Track learning progress  
* View quiz scores  
* Monitor video watch time  
* Add lessons manually  
* Use an AI assistant to generate new lessons  
* Manage subjects and quizzes  
* Manage the reward shop  
* Export all academic history as a backup file

---

## **Core User Roles**

## **1\. Student Portal**

The student portal should be distraction-free, fun, and easy to use.

The student should log in and immediately see what they need to do next. The dashboard should clearly show today’s lessons, progress, points, streaks, upcoming Friday quiz, and suggested review topics.

The student should be able to:

* View today’s learning missions  
* Pick a subject

* Get help if they struggle

* Complete a weekly Friday review quiz

The student experience should feel motivating and visual, not like a school admin system.

* Encouraging language
